## Frontend Documentation

The documentation is built using Docusaurus and can be found in the `frontend-book` directory. To run the documentation site locally:

```bash
cd frontend-book
npm install
npm run start
```

This will start the development server and open the documentation in your browser.

## About This Project

This project follows a spec-driven development approach, with detailed planning and task breakdowns stored in the `specs/` directory. Each feature has its own specification, plan, and task files to ensure comprehensive coverage of requirements.
