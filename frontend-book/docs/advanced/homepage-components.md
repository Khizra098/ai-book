# Homepage Components

This document provides information about the custom components used in the homepage redesign for the Physical AI & Humanoid Robotics book.

## HeroSection Component

The HeroSection component displays the main hero area of the homepage with the humanoid robot image, title, and subtitle.

### Usage
The component is automatically included in the homepage and requires no additional configuration.

### Features
- Responsive design that adapts to different screen sizes
- Humanoid robot image display
- Title and subtitle text
- Accessibility features with proper alt text

## ChapterNavigation Component

The ChapterNavigation component provides a dropdown menu for navigating through book chapters.

### Usage
The component is automatically included in the homepage and requires no additional configuration.

### Features
- Green "Start Book" button
- Dropdown menu with chapter links
- Click outside to close functionality
- Keyboard accessible navigation

## ModuleCards Component

The ModuleCards component displays a grid of modules with titles, descriptions, and links.

### Usage
The component is automatically included in the homepage and requires no additional configuration.

### Features
- Responsive grid layout
- Hover effects on cards
- Module titles and descriptions
- Action buttons with links

## Styling

All components use CSS modules for styling to ensure encapsulation and avoid style conflicts. Additional global styles are defined in `src/css/homepage.css` for responsive behavior and accessibility features.
